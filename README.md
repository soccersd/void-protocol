## 🎯 Project Overview

**VOID-PROTOCOL** is a complete miner supplier concept designed for the **Club HashCash ecosystem**. We deliver a coherent brand identity, three-tier hardware lineup, and production-ready website that demonstrates readiness for ecosystem integration.

### ✨ Key Features

- 🛠️ **Three-tier hardware lineup** - Fragment-01, Fissure-05, Anomaly-X
- 🎨 **Industrial Noir design** - Tactical precision with aggressive purple glow effects
- 🛰️ **Real-time mission control** - Event Horizon dashboard with live network monitoring
- 🚀 **Production-ready** - Full Next.js website optimized for deployment

## 🚀 Quick Start

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/void-protocol.git
cd void-protocol

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

> 💡 Open [http://localhost:3000](http://localhost:3000) to view the site
